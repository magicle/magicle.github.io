Mailet
    Social Networking by Email Proxying

Download Facet Source Code:

Facet 1.0.0: github repository
git clone https://github.com/magicle/Mailet.git

Prerequisite:
(a) Linux operating system, Mailet implementation is tested on Ubuntu 12.04
(b) Email application, such as Thunderbird, or web email service
(c) Acknowledgement: Facet server is based on the following open source projects:

FastGC: a secure two-party computation framework
TwitterAPI: a Python wrapper for Twitter.com API
Facet repository includes a installer for these projects.

Install:

./configure
make
make install
